--- layout: scd title: 1.4.8.1.4 trackLanguage permalink: /scd/tracklanguage scd-label: Track Language scd-def: The primary language(s) sung or spoken in the track. scd-imp: Optional, Repeatable --- Comment Enter a separate trackLanguage element for each language sung or spoken in the track. Value Control

Use the ISO 639-2 Alpha-3 code for the language. When a bibliography code and a terminology code are provided, use the terminology code (the 2nd code listed in the pair at the following link).

https://www.loc.gov/standards/iso639-2/php/English_list.php

Example eng; slk
