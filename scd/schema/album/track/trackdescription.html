--- layout: scd title: 1.4.8.1.3 trackDescription permalink: /scd/trackdescription scd-label: Track Description scd-def: A free-text description of the track that gives an overview of the artists involved in the specific track if specified, the genre of the particular track, and any defining features. This element may also be used to record information regarding the original artist and song. scd-imp: Optional --- Comment This element will be utilized for user-side display. Value Control None
