--- layout: scd title: 1.8.4.1 imageID permalink: /scd/imageIdentifier scd-label: Image Identifier scd-def: An unambiguous reference to the image. scd-imp: Mandatory within image container element. --- Comment Image identifier is also the same as the file name. Value Control

scd_YYYYMMDD_000.ext

Each identifier starts with “scd” and ends in a numerical reference to the date taken and number order taken, followed by the image format extension.

Example scd_20191125_001.jpg
