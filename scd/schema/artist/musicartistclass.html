--- layout: scd title: 1.6.1.2 musicArtistClass permalink: /scd/musicartistclass scd-label: Music Artist Class scd-def: The class of artist in relation to the creation and recording of the album. scd-imp: Mandatory if using musicArtist, Repeatable --- Value Control

solo artist; guest artist; group member

The value “solo artist” is to be used if there is a single artist responsible for the album and does not require a 1.6.1.3 musicArtistRole element to refine. See Appendix B for more information.
